# AcneAdvisor 💊

Java 콘솔 기반 여드름 분석 및 피부 관리 추천 프로그램입니다.  
사용자가 여드름 위치, 개수, 통증 여부 등을 입력하면  
→ 여드름 유형을 자동으로 분석하고  
→ 추천 약과 관리 팁을 안내해 줍니다.

---

## 🧩 주요 기능

- 사용자 피부 상태 입력
- 여드름 유형 자동 판별
- 적절한 약/제품 추천
- 생활 관리 팁 안내
- 구조화된 클래스 기반 설계

---

## 🔧 사용 기술

- Java (콘솔 앱)
- 객체지향 클래스 설계
- GitHub 버전 관리

---

## 📁 클래스 구조

| 클래스 이름 | 설명 |
|-------------|------|
| `User` | 사용자 정보 (이름, 나이, 피부 타입) |
| `AcneStatus` | 여드름 상태 정보 |
| `Advisor` | 여드름 분석 및 추천 로직 |
| `Recommendation` | 추천 정보 저장 및 출력 |
| `Main` | 실행 흐름 제어 (입력 → 분석 → 출력) |

---

## 📌 실행 방법

1. Java IDE에서 프로젝트 열기 (예: IntelliJ)
2. `Main.java` 실행
3. 안내에 따라 여드름 정보를 입력
4. 분석 결과 확인

---

## 🙋 만든 이유

청소년들이 겪는 여드름 고민을  
간단하게 분석하고 관리할 수 있도록 돕는  
실용적인 학습형 프로젝트입니다.
